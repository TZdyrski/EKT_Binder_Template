# Try it out!
To test this out, determine the URL of a notebook you want to visit.
Note: if you are trying to view a notebook on Github, ensure you are
using the URL of the raw content (viewable by clicking `Raw`; it should
start with `raw.githubusercontent.com`).
Then, double percent-encode this URL,
Finally, visit
`https://mybinder.org/v2/gh/tzdyrski/ekt_binder_template/HEAD?urlpath=pluto/open?url=<double percent-encoded URL>`
